## 🚀 How to Run Notebooks

**Prerequisites:** Python 3.x and `pip` installed.

```bash
# 1. Clone this repository
git clone https://github.com/abdulmutallib990-netizen/python.git
cd python

# 2. Install Jupyter
pip install notebook

# 3. Launch Jupyter
jupyter notebook
```

Then navigate into `notebooks/`, `practice_problems/`, or `assignments/` and open any `.ipynb` file.
